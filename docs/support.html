Before You Report

  1. Restart the app and try the same steps one more time.
  2. Confirm your installed build is your latest available version.
  3. Check the issue tracker to avoid duplicate reports.

Include These Details

  • Device model (for example, iPhone 14 Pro).
  • Operating system version (for example, iOS 18.3).
  • Game version or build number.
  • Exact reproduction steps.
  • Screenshots or screen recordings, if available.
Faster triage tip: issues with exact steps and device/build info are resolved significantly faster than generic reports.

Common Questions

Does the game need internet?

No. Multiply Rush is designed for offline gameplay.

Does the game use ads or tracking?

No. The project is built with an offline-first, no-ads approach.

Where is my data stored?

Your settings and progression are stored locally on your device.
