Support

Need help with Multiply Rush? Use the issue tracker for bug reports, support requests, and feedback.

Open a Support Request Browse Existing Issues

Before You Report

  1. Restart the app and try again.
  2. Confirm your app version is the latest build you installed.
  3. Check whether the same issue is already reported in the issue tracker.

Include These Details

  • Device model (for example, iPhone 14 Pro).
  • OS version (for example, iOS 18.3).
  • Game version/build number.
  • Clear reproduction steps.
  • Screenshots or screen recording, if available.

Common Questions

Does the game need internet?
No. Multiply Rush is designed for offline gameplay.

Does the game use ads or tracking?
No. The project is built with an offline-first, no-ads approach.

Where is my data stored?
Your settings and progression are stored locally on your device.
